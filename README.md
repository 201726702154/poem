# lmz
基于安卓系统开发的诗人修炼手册APP

诗人修炼手册的功能包括：
A.每日推荐；B.诗词闯关；C.才思创作；D.搜索；E.分类；F.语音播放、
G.识别以及测试；H.日间/夜间模式；I.滑动翻页；J.用户还可以对诗歌点赞、收藏和分享等。
